# QR Codec V1 (Matrix Parser)

Легковаговий об'єктно-орієнтований модуль для просторового кодування та декодування QR-кодів Версії 1 (21×21 піксель). Проєкт фокусується на базовій топології стандарту: "зигзагоподібному" обході матриці, парсингу бітового потоку та екстракції даних в ідеальних умовах (без застосування поліномів корекції помилок).

## Основні характеристики

* **Двостороннє перетворення:** Модуль містить як кодер (`encode`) для генерації матриці з тексту, так і парсер (`decode`) для розшифровки.
* **Просторова маршрутизація:** Ітератор автоматично оминає службові зони (Finder Patterns та Timing Patterns) під час зчитування/запису.
* **Підтримка протоколу:** Реалізовано роботу з байтовим режимом (Byte Mode, маркер `0100`).
* **Чиста архітектура:** Код написаний з використанням `Type Hinting`, генераторів (`yield`) та не має залежностей від сторонніх бібліотек (Zero Dependencies).
