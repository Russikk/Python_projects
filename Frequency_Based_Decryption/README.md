# Frequency-Based Caesar Cipher Decryption

## Опис проекту
Програмна реалізація інструменту для автоматичного криптоаналізу та зламу шифру Цезаря на основі частотного аналізу (без використання ключа). 

## Архітектура алгоритму
Алгоритм використовує метод повного перебору простору ключів (Brute Force у діапазоні $0 \le k \le 25$) з подальшою евристичною оцінкою кожного результату. 

Для визначення правильного ключа застосовується статистичний критерій **Пірсона (Хі-квадрат, $\chi^2$)**. Алгоритм порівнює частотний розподіл літер у кожному варіанті розшифрування з еталонним розподілом літер англійської мови. Варіант з мінімальним значенням $\chi^2$ приймається як валідний відкритий текст (Plaintext).

## Особливості реалізації
- **Збереження регістру:** Модульна арифметика виконується відносно динамічної бази ASCII (`65` для Uppercase, `97` для Lowercase).
- **Фільтрація:** Спецсимволи, цифри та пробіли ігноруються під час математичного зсуву та статистичного підрахунку, повертаючись у вихідний потік без змін.
- **Масштабованість:** Аналіз частот виконується відносно загальної довжини алфавітних символів у конкретному тексті (динамічний Expected Count).
