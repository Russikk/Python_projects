# AES-256 Cryptography: CFB Mode (Python Implementation)

Цей репозиторій містить еталонну реалізацію симетричного алгоритму шифрування **AES-256** з використанням режиму **CFB (Cipher Feedback)**. Проєкт розроблено на Python з використанням бібліотеки `cryptography`.

##  Архітектурні особливості режиму CFB
На відміну від стандартного CBC (Cipher Block Chaining), який працює виключно цілими блоками по 128 біт, режим CFB перетворює AES на **потоковий шифр (stream cipher)**.

* **Відсутність Padding:** Даний режим дозволяє шифрувати потоки даних довільної довжини без необхідності доповнювати їх до кратності блоку (наприклад, алгоритмом PKCS7).
* **Синхронізація:** Вимагає унікального вектора ініціалізації (IV) для кожної криптографічної сесії. Якщо при передачі даних пошкоджується один біт шифротексту, це впливає лише на поточний та наступний блоки, після чого система відновлює синхронізацію.
